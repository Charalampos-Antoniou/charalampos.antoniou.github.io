Bank Loan Analysis in Tableau

This project on Bank Loan Data consists of three interactive dashboards designed to analyze and monitor the bank's lending activities. The first dashboard provides a comprehensive Bank Loan Report, highlighting key loan metrics and tracking performance over time. It helps the bank assess the health of its loan portfolio and make data-driven decisions. The second dashboard uses various chart types to visualize loan-related metrics and trends, offering insights into different loan parameters for more informed decision-making. The third dashboard consolidates all essential loan data, offering a Details Dashboard with a holistic snapshot of borrower profiles and loan performance, enabling efficient access to critical information.

Adidas US Sales Analysis in PowerBI

This Power BI project features a single interactive dashboard that provides a comprehensive view of sales across various dimensions. The dashboard displays monthly sales trends, allowing users to track performance over time. It also breaks down sales by state, region, and product, giving a detailed understanding of which products perform best in specific areas. Additionally, sales are segmented by retailer, providing insight into which retail partners drive the most revenue. The dashboard also includes the operating margin, enabling an analysis of profitability across different products and regions. This Power BI dashboard offers a complete overview of Adidas' US sales performance, supporting strategic decision-making.
